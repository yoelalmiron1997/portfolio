QA Automation API Portfolio

Automatización sobre una API REST, enfocada en análisis de riesgos, diseño de casos de prueba, validación de respuestas y generación de reportes.

Robot Framework · Python · RequestsLibrary · Swagger · GitHub Actions
Repositorio Deploy pendiente de publicación
Contexto
Proyecto personal para consolidar en un solo repositorio un flujo completo de automatización de pruebas sobre una API REST pública, desde el análisis inicial hasta la integración continua.
Objetivo
Automatizar la validación funcional de los endpoints principales de la API, integrar la ejecución en un pipeline de CI y dejar documentado el razonamiento detrás de cada caso de prueba.
Riesgos considerados
Cambios en el contrato de la API que rompan las pruebas silenciosamente, dependencia de datos externos inestables, y falta de reportes claros ante una ejecución fallida en CI.
