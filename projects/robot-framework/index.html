Robot Framework Test Automation – Level 2

Resumen del curso realizado en Udemy, habilidades reforzadas, certificado y aplicación práctica mediante una demostración sencilla.

Robot Framework · Python ·
Repositorio Ver certificado
Contexto
Curso de Udemy orientado a profundizar el diseño, la organización y el mantenimiento de pruebas automatizadas con Robot Framework, como continuación de un primer nivel introductorio.
Objetivo
Reforzar buenas prácticas de estructuración de test suites, manejo de keywords personalizadas, uso de librerías externas y organización de recursos reutilizables, y aplicar esos conceptos en una demostración práctica propia.
Riesgos considerados
No aplica: proyecto de tipo formativo.
Estrategia de pruebas
La demostración práctica del curso implementa un caso de prueba sencillo de interfaz web con SeleniumLibrary, organizado en keywords reutilizables y separado en recursos y test suites según buenas prácticas de Robot Framework.
Tecnologías
Robot Framework Python SeleniumLibrary
Evidencia
Evidencia pendiente de publicación
Resultados
Próximamente se agregarán nuevos casos.
Aprendizajes
Organizar pruebas en recursos reutilizables reduce el mantenimiento a largo plazo y facilita que otra persona del equipo entienda la suite sin depender del autor original.
