Comparador de configuraciones JSON

Herramienta en Python para comparar archivos de configuración JSON y generar un reporte HTML visual con sus diferencias.

Python · JSON · HTML · CSS
Repositorio Deploy
Contexto
En proyectos con múltiples entornos (desarrollo, staging, producción) es común que las configuraciones diverjan sin que nadie lo note hasta que provoca un incidente. Esta herramienta nació para hacer esas diferencias visibles y fáciles de revisar.
Objetivo
Construir una utilidad de línea de comandos en Python que reciba dos archivos JSON, calcule sus diferencias (claves agregadas, eliminadas y modificadas) y genere un reporte HTML legible para compartir con el equipo.
Riesgos considerados
Estructuras JSON anidadas de forma profunda, diferencias de tipo de dato entre valores aparentemente iguales, y reportes poco legibles cuando el volumen de diferencias es alto.
Estrategia de pruebas
Casos de prueba con archivos JSON de ejemplo que cubren adiciones, eliminaciones, cambios de valor y cambios de tipo, verificando que el reporte generado refleje correctamente cada categoría de diferencia.
Tecnologías
Python JSON HTML CSS
Evidencia
Evidencia pendiente de publicación
Resultados
Próximamente se agregarán nuevos casos.
Aprendizajes
Un reporte de diferencias es tan útil como legible: la parte más difícil no fue calcular el diff, sino presentarlo de forma que alguien lo entienda sin explicación adicional.
